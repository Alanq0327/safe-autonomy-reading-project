# safe-autonomy-reading-project
# Independent Reading Project: Real-Time Safeguard and Uncertainty-Aware Autonomy

This repository documents my independent reading project on safe autonomous systems, with a focus on Operational Design Domain (ODD), robust perception, real-time perception safety assessment, observation-robust decision-making, and system-level risk assessment.

The project was developed as part of my research preparation for future Ph.D. study in safe autonomous systems, uncertainty-aware robotics, and real-time safeguard mechanisms for robotic and cyber-physical systems.

## Motivation

Modern autonomous systems must operate under uncertain, dynamic, and interactive real-world conditions. In such environments, safety cannot be guaranteed only by improving perception accuracy or designing a nominal controller. Instead, reliable autonomy requires a system-level framework that connects:

- operating-domain definition,
- perception robustness,
- runtime reliability assessment,
- robust decision-making,
- and risk-aware system monitoring.

This reading project aims to understand how these components can be integrated into a broader safe autonomy research direction.

| 5 | *Risk Assessment in Autonomous Driving: A Comprehensive Survey* | System-level risk assessment | Provides a broader taxonomy of risk sources, methods, data, and system architectures |
